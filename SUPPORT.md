# Support

For help using an individual tool, first review the information on that tool's page at [All Tools Verse](https://alltoolsverse.com/tools/).

- Broken directory link or incorrect listing: open a GitHub issue.
- New tool suggestion: use the tool-suggestion issue form.
- General contact: `contact@alltoolsverse.com`.
- Sensitive security report: follow [SECURITY.md](SECURITY.md).

Please do not include confidential files, personal data, passwords or access tokens in public issues.
