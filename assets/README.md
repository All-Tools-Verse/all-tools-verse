# Brand assets

This folder contains official All Tools Verse brand assets used to identify the repository.

- `all-tools-verse-icon.png`: square indigo brand icon.
- `all-tools-verse-logo.png`: indigo logo and dark wordmark for light interfaces.
- `all-tools-verse-logo-dark.png`: matching logo with a light wordmark for dark interfaces.
